# Parking-sharing-mini-program
还在为停车发愁？这款停车共享小程序来帮忙！它整合周边小区、写字楼、商场等闲置车位资源，打开小程序，输入目的地，即可精准查看附近可用共享车位，空闲数量、收费标准一目了然。地图导航功能，助你快速抵达。支持提前预约，避免“跑空”。个人闲置车位也能发布出租，增加额外收益。缓解停车难题，让停车变得轻松简单 。
